1. ถ้าต้องการทำนายล่วงหน้า "3 วัน"
ค่า PREDICTION_STEPS จะเปลี่ยนไปตาม Interval ดังนี้:

กรณีใช้ INTERVAL = "1d" (รายวัน):

1 แท่ง = 1 วัน

ใส่ PREDICTION_STEPS = 3 (3 แท่ง × 1 วัน = 3 วัน) ✅ วิธีที่ง่ายที่สุด

กรณีใช้ INTERVAL = "1h" (รายชั่วโมง):

สำหรับหุ้น (ตลาดเปิด ~7 ชม./วัน): 1 วัน ≈ 7 แท่ง → 3 วัน ≈ 21 Steps

สำหรับ Crypto (24 ชม.): 1 วัน = 24 แท่ง → 3 วัน = 72 Steps

กรณีใช้ INTERVAL = "15m" (ราย 15 นาที):

1 ชั่วโมงมี 4 แท่ง

สำหรับหุ้น: 1 วัน ≈ 26 แท่ง (6.5 ชม.) → 3 วัน ≈ 78 Steps

2. ถ้าต้องการทำนายล่วงหน้า "3 ชั่วโมง"
กรณีใช้ INTERVAL = "1h" (รายชั่วโมง):

1 แท่ง = 1 ชั่วโมง

ใส่ PREDICTION_STEPS = 3 ✅

กรณีใช้ INTERVAL = "15m" (ราย 15 นาที):

1 ชั่วโมง = 4 แท่ง (15นาที × 4)

3 ชั่วโมง = 4 × 3 = 12 Steps ✅ แนะนำอันนี้จะได้รายละเอียดดีกว่า

กรณีใช้ INTERVAL = "1d":

❌ ทำไม่ได้ เพราะ 1 แท่งมันกว้างกว่า 3 ชั่วโมง (ทำนายได้หยาบที่สุดคือ 1 วัน)

3. ถ้าต้องการทำนายล่วงหน้า "3 นาที"
กรณีใช้ INTERVAL = "1m" (รายนาที):

1 แท่ง = 1 นาที

ใส่ PREDICTION_STEPS = 3 ✅

❌ ทำไม่ได้ เพราะ Interval มันหยาบเกินกว่าจะมองเห็น 3 นาทีครับ
